# Netflix Data Cleaning and Preprocessing

## Objective

The objective of this project is to clean and preprocess a Netflix dataset to make it suitable for analysis.

## Tools Used

* Python
* Pandas
* Visual Studio Code

## Dataset

Netflix Movies and TV Shows Dataset

## Data Cleaning Steps Performed

1. Loaded the dataset using Pandas.
2. Checked for missing values.
3. Checked for duplicate records.
4. Removed duplicate records.
5. Standardized column names.
6. Cleaned text fields by removing extra spaces.
7. Converted the date_added column into datetime format.
8. Verified data types.
9. Saved the cleaned dataset.

## Results

* Total Records: 8790
* Missing Values Found: 0
* Duplicate Records Found: 0
* Cleaned Dataset Generated Successfully

### Note

Dataset inspection showed no missing values and no duplicate records. The preprocessing focused on:

* Column name standardization
* Text formatting cleanup
* Date conversion
* Data type verification
* Dataset validation
